Node app to Problem 1 of Round 2
Problem: Given a set of product names(separated by new line), you have to return average price of first 20 products in downloadable CSV format from Amazon.in. 
Resource: 
1. Use cheerio(optional, you can use if any other npm that works better) to scrape the data and make URL using sample product keywords given below. 
Sample Product Names:
Running Shoes
Treadmills
Dumbbells
Home Gym Equipment
Weight Lifting Shoes
Weight lifting Belts
Workout Headphones

Expected Output:
CSV file with product name along with average price.

Note:
1.Please install dependencies by given command
npm install --save
2.After Installing run the app
node app.js
